About the API catalogue

If you want to access or transfer data, you can check if there’s an existing way to access and transfer data across HMRC.

By reusing what’s already available, you don’t need to build a single use solution which saves you time and money.

There are two ways you can securely access and transfer data across HMRC.

  1. APIs
  2. File transfer connections

APIs let IT systems and applications talk to each other and share data in real time.

File transfers move batch files or sets of data from A to B. Unlike an API, a file transfer only moves data from one place to another.

File transfers use connections to send data. You can reuse existing file transfer connections for your service.

Review available HMRC APIs or check for file transfer connections

Using the service

Use this service to:

  • get contact information for specific APIs or file transfer connections
  • find existing HMRC APIs and file transfer connections
  • read details about APIs

Read our get started guide before you start searching.

Reusing APIs and file transfer connections

Instead of building a new API or file transfer connection every time your service needs to access or transfer data, you can reuse existing APIs and file transfer connections.

Reusing what's available:

  • saves time and money
  • improves flexibility, governance, speed and usability
  • encourages change and innovation
  • removes the need for single use APIs or file transfers
