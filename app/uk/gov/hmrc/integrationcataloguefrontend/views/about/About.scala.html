About the API catalogue

If you want to access or transfer data, you can check if there’s an existing way to access and transfer data across HMRC.

By reusing what’s already available, you don’t need to build a single use solution which saves you time and money.

There are two ways you can securely access and transfer data across HMRC.

  1. APIs
  2. File transfer flows

APIs let IT systems and applications talk to each other and share data in real time.

File transfer flows transfer batch files or sets of data from A to B. Unlike an API, a file transfer flow only moves data from one place to another.

Each file transfer flow uses a specific file transfer pattern. You can reuse existing file transfer patterns to build your own file transfer flow to get data for your service.

Using the service

Use this service to:

  • search for the data you need
  • find existing HMRC APIs and file transfers that meet your service’s needs
  • get contact information for specific APIs or file transfer flow
  • view details about APIs and file transfers

Read our get started guide before you start searching.

Reusing APIs and file transfer patterns

Instead of building a new API or file transfer flow every time your service needs to access or transfer data, you can reuse existing APIs and file transfer patterns.

Benefits of reuse

  • Greater flexibility, governance, speed and usability
  • Saves time, effort and money
  • Encourages change and innovation
  • APIs and file transfers are fit for the future
  • Removes the need for single use APIs or file transfers
